# HackerDesktop
HackerDesktop - кастомизация рабочего стола с помщью Winstep Nexus и Rainmeter

## Post-Install
После установки нажми Ctrl+F11 или пкм по доку, NeXuS -> Настройки -> Дополнительно -> Возвратить и выебри файл HackerDesktop.wbk
